# kdniao
快递鸟 SDK Go 语言版

## Requirements
Go 1.12 or above.

## Install
```shell
go get -u -x -v -insecure github.com/ttlv/kdniao
```

### Configure Parameter
EBusinessID 和 AppKey 去 [快递鸟用户管理后台](http://kdniao.com/UserCenter/UserHome.aspx) 查看

```shell
shell> vi config.txt
EBusinessID,AppKey
```
